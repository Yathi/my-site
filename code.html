How to install a Python library

You have to install a python package in your system before you can import it. Or else it will give you an ImportError at runtime. To install follow the steps mentioned below:

  • Go to Python PI and search for the package you want to install. It will usually be in .tar.gz format.
  • Download the tar and extract it.
  • Navigate inside and you find a setup.py
  • Do a python setup.py build
  • If the above command ran successfully do a sudo python setup.py install

And that is it. Your package is istalled :)
